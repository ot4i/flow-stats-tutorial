MessageFlow Statistics

Learn how turn on messageflow statistics tracking and view them in the WebUI.
OVERVIEW

This tutorial shows how a user would go about enabling messageflow statistics tracking and then monitor said statistics in real-time. The example uses a simple message flow to demonstrate some new usefule techniques.

The server.conf.yaml file will be introduced as a useful way for tweaking the behaviour of an integration server. In this example we will only enable messageflow statistics tracking.

Finally, the user will make several requests to the server before being guided through accessing the WebUI of said server. They will then be shown where to find the statistics in the WebUI.

This tutorial takes approximately 15 minutes.
Tutorial Topics
  • Finding and editing the server config file
  • Manually enabling messageflow statistics tracking
  • Accessing the WebUI
  • Viewing the statistics panel
Learning outcomes
  • Understand how to locate the server config file, turn on messageflow statistics tracking and monitor them in real time using the WebUI.
